# 题库自动清洗规则说明文档

## 一、概述

本系统内置题库自动清洗工具，支持对未结构化的纯文本题库进行智能识别，自动提取题型、题干、选项、答案和解析字段。清洗后的数据可直接导入数据库使用。

## 二、支持的原始题库格式

### 2.1 题型标记识别规则

系统依次扫描每一行，匹配以下模式来判断当前题目所属题型：

| 题型 | 可识别的标记模式 |
|------|-----------------|
| 单选题 | `单选题` `单项选择题` `一、单选题` `一、单项选择题` `一、 单项选择` |
| 多选题 | `多选题` `多项选择` `二、多选题` `二、多项选择题` `二、 多项选择` |
| 判断题 | `判断题` `三、判断题` `三、 判断` `判断` |
| 简答题 | `简答题` `四、简答题` `论述题` `四、 简答` `简答` `论述` |

**规则**：题型标记支持大小写不敏感匹配。遇到新的题型标记时，后续题目自动归类到该题型，直到遇到下一个题型标记。

### 2.2 题干识别规则

- **题号格式**：行首包含数字 + 标点 + 空格 + 内容的模式，如：
  - `1. 题干内容`
  - `1、题干内容`
  - `1）题干内容`
  - `1) 题干内容`
- **注意**：题号后必须有至少一个非空白字符，否则不识别为题干。

### 2.3 选项识别规则

- **格式要求**：每行以单个大写字母 + 标点 + 空格 + 内容开头：
  - `A. 选项内容`
  - `B、选项内容`
  - `C．选项内容`（全角点号）
- **支持的字母范围**：A-H（最多8个选项）
- **收集逻辑**：从题干下一行开始，连续收集所有匹配选项规则的文本行，直到遇到非选项行或下一题开始。

### 2.4 答案识别规则

- **标记格式**：
  - `答案：A`
  - `参考答案：ABC`
  - `答案：正确`（判断题）
  - `答案：错误`（判断题）
- **判断题特殊处理**：
  - 如果选项只有2个（通常是"正确/错误"或"A/B"），且答案包含"对"/"正确"或字母"A"，标准化为"正确"
  - 如果答案包含"错"/"错误"或字母"B"，标准化为"错误"

### 2.5 解析识别规则

- **标记格式**：
  - `解析：解析内容...`
  - `答案解析：解析内容...`

## 三、自动推断规则

### 3.1 多选题推断
当当前题型为"单选题"，但答案包含多个字母（如"ABC"），系统自动推断为"多选题"并在预览中标注。

### 3.2 简答题推断
当题目没有识别到任何选项时，自动推断为"简答题"。

## 四、导入校验规则

### 4.1 校验状态

| 状态 | 条件 | 说明 |
|------|------|------|
| ✅ 正常(valid) | 题干、答案均正常识别 | 可直接导入 |
| ⚠️ 警告(warning) | 未找到答案或解析 | 可导入，但建议人工确认 |
| ❌ 错误(error) | 题干为空 | 无法导入，需修正或删除 |

### 4.2 自动去重
导入时自动比对题库中已有题目的题干内容，完全相同的题目自动跳过，避免重复导入。

## 五、使用建议

1. **格式规范化**：建议在导入前尽量统一原始题库格式，如：
   - 确保题型分节标题正确
   - 每题独占一行题号开头
   - 选项统一使用「字母+点号」格式

2. **逐步导入**：大批量题库建议分批导入，每次100-200题，便于检查识别质量。

3. **人工复核**：导入预览阶段仔细检查标有⚠️警告的题目，必要时手动修改。

4. **备份原始数据**：导入前保留原始题库文件备份，以防万一。

## 六、示例

### 输入示例
```
一、单选题

1. 马克思主义诞生的标志是？
A. 《共产党宣言》的发表
B. 巴黎公社的成立
C. 俄国十月革命的胜利
D. 中国共产党的成立
答案：A
解析：1848年2月，《共产党宣言》的发表标志着马克思主义的诞生。
```

### 识别结果
- 题型：单选题
- 题干：马克思主义诞生的标志是？
- 选项：["《共产党宣言》的发表", "巴黎公社的成立", "俄国十月革命的胜利", "中国共产党的成立"]
- 答案：A
- 解析：1848年2月，《共产党宣言》的发表标志着马克思主义的诞生。
- 状态：✅ 正常
